== Description ==

Submit Content is a free and open source WordPress plugin maintained by the bharat thapa that allows users to submit various types of content from frontend of the WordPress website.

Submit Content is a free and open source WordPress plugin, and will be fully supported and maintained as long as is necessary.

At a glance, this plugin allows the following:

* Administrators can choose whether only logged in users can submit content or site visitors as well.
* Administrators can also allow visitors to register during content submission (if visitors can submit is enabled in the plugin settings page).

== Frequently Asked Questions ==

= Default settings =

By default, the submitted content will be saved as a post type 'post' in the database. But you can change this default behavior by visiting plugin settings page.
